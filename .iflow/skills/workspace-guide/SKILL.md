# Welcome to iFlow

Hi, this is iFlow (心流). It\'s an alternative to Claude\'s cowork. It helps you work on your files with AI and automate the mundane tasks so you don\'t have to.

Before we start, use the question tool to ask:
"Are you more technical or non-technical? I\'ll tailor the explanation."

## If the person is non-technical
iFlow feels like a chat app, but it can safely work with the files you allow. Put files in this workspace and I can summarize them, create new ones, or help organize them.

Try:
- "Summarize the files in this workspace."
- "Create a checklist for my week."
- "Draft a short summary from this document."

## Skills
Skills add new capabilities. We can add them later when you\'re ready.

## If the person is technical
iFlow is a GUI for iFlow CLI. Everything that works in iFlow CLI works here.

Most reliable setup today:
1) Install iFlow CLI from https://cli.iflow.cn
2) Configure providers there (models and API keys)
3) Come back to iFlow and start a session

Skills:
- Install from the Skills tab, or add them to this workspace.
- Docs: https://platform.iflow.cn/cli/examples/skills


MCP servers:
- Add external tools via settings.json.
- Docs: https://platform.iflow.cn/cli/examples/mcp

Config reference:
- Docs: https://platform.iflow.cn/cli/configuration/settings

End with two friendly next actions to try in iFlowork.
